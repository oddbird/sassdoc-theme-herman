$brand-colors (map)

scss
$brand-colors: (
  'brand-orange': hsl(24, 100%, 39%),
  'brand-blue': hsl(195, 85%, 35%),
  'brand-pink': hsl(330, 85%, 48%),
);

Description

We use bright primary colors for the main brand, everything else is based on these colors.

Map Properties

'<color-name>': (Color | String | List)

Each <color-name> key can be assigned a color or previously-defined key, followed by an optional map of adjustments (as defined in Accoutrement-Color).

  'yield': 'brand-orange',
  'stop': hsl(0, 85%, 50%),

These status colors for UI elements give a quick sense of progress, warning, or errors.
