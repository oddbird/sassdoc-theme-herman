Herman Banner Region

Styles related to the top banner in generated Herman docs.

@mixin banner-arrow()

Apply a bottom-border with centered down-arrow triangle, for use as a divider between the banner and main content.

Examples

scss 
[data-region='banner'] {
  @include banner-arrow;
}
css compiled 
[data-region='banner'] {
  border-bottom: 8px solid #0d7fa5;
  border-bottom-color: var(--color-theme-dark, #0d7fa5);
  box-shadow: 0 2px 2px rgba(59, 64, 66, 0.25);
  display: block;
  position: relative;
  width: 100%;
}

[data-region='banner']::before, [data-region='banner']::after {
  content: "";
  border: 0.675rem solid transparent;
  border-top-color: #0d7fa5;
  border-top-color: var(--color-theme-dark, #0d7fa5);
  border-width: 0.675rem 1.35rem 0;
  left: 50%;
  position: absolute;
  top: 100%;
  transition: border-width 200ms;
}

[data-region='banner']::before {
  content: "";
  margin-top: 8px;
  transform: translateX(-50%) translateY(-2px);
}

[data-region='banner']::after {
  content: "";
  border-top-color: #fff;
  border-top-color: var(--color-background, #fff);
  transform: translateX(-50%) translateY(-4px);
}
html 
<header data-region="banner">
  Hello World
</header>

[data-region='banner']

scss 
[data-region='banner'] {
  @include banner-arrow;
  @include z-index('banner');
  align-items: center;
  display: flex;
  padding: size('shim');
}

Layout styles for Herman’s top-banner region.

Example

njk 
{% import 'utility.macros.njk' as utility %}
<header data-region="banner">
  <button data-nav-toggle>
    {{ utility.icon('menu', size='nav-icon') }}
  </button>
  <div class="project-title">
    <a href="index.html" rel="home" class="project-name">Herman</a>
    <span class="project-version">1.0.0</span>
  </div>
</header>
html compiled 
<header data-region="banner">
  <button data-nav-toggle>
    <svg data-icon="menu" data-icon-size="nav-icon">
      <use xlink:href="#icon-menu" />
    </svg>
  </button>
  <div class="project-title"> <a href="index.html" rel="home" class="project-name">Herman</a>
 <span class="project-version">1.0.0</span>

  </div>
</header>

.project-title

scss 
.project-title {
  align-items: baseline;
  display: flex;
  margin-left: auto;
  margin-right: auto;
}

Used for the top-banner project title, including both the name and version of the project.

Example

html 
<div class="project-title">
  <a href="index.html" rel="home" class="project-name">Herman</a>
  <span class="project-version">1.0.0</span>
</div>

.project-name

scss 
.project-name {
  font-size: size('h1');
  font-weight: bold;
  padding-left: size('half-shim');
  padding-right: size('half-shim');

  #{$link} {
    text-decoration: none;
  }
}

Specific styling for the top-banner project name.

Example

html 
<a href="index.html" rel="home" class="project-name">Herman</a>

.project-version

scss 
.project-version {
  @include font-family('sans');
  color: color('text-light');
  color: var(--color-text-light, color('text-light')); //sass-lint:disable-line no-duplicate-properties
  font-size: size('h3');
}

Less prominent text for the top-banner project version.

Example

html 
<span class="project-version">1.0.0</span>

[data-nav-toggle]

scss 
[data-nav-toggle] {
  color: color('action');
  color: var(--color-action, color('action')); //sass-lint:disable-line no-duplicate-properties
  padding: size('half-shim');

  @include above('page-break') {
    padding-left: size('shim');
    padding-right:  size('shim');
  }

  @include above('nav-break') {
    display: none;
  }

  #{$focus},
  &[aria-pressed='true'] {
    color: color('focus');
    color: var(--color-focus, color('focus')); //sass-lint:disable-line no-duplicate-properties
    outline: 0;
  }
}
