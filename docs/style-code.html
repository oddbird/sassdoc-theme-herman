Herman Code-Block Styles

Code-block styles and syntax highlighting, for both user and machine-generated code samples.

Code Blocks

scss 
.code-block {
  align-items: stretch;
  display: flex;
  flex: 1 1 20em;
  flex-direction: column;
  max-height: size('rhythm') * 12;
  max-width: 100%;
  overflow: hidden;
}

.code-header

scss 
.code-header {
  @include contrasted('theme-light');
  flex: 0 0 auto;
  padding: 0 size('shim');
}

.code-description

scss 
.code-description {
  @include before(' – ');
}

.hljs-pre, .text-block pre

scss 
.hljs-pre,
.text-block pre {
  background-color: color('slight');
  border: 1px solid color('callout');
  color: color('code');
  flex: 1 1 auto;
  flex-wrap: wrap;

  code {
    padding: size('shim');
  }
}

$hljs-colors

scss 
$hljs-colors: (
  'hljs-comment': #93a1a1,
  'hljs-green': #859900,
  'hljs-cyan': #2aa198,
  'hljs-blue': #268bd2,
  'hljs-yellow': #b58900,
  'hljs-orange': #cb4b16,
  'hljs-red': #dc322f,
  'hljs-formula': #eee8d5,
);

Colors for code-highlighting with hljs, based on Solarized-Light & the Herman brand.
