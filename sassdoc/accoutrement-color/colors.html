@function color()

Description

Access any color in your palette, or generate new colors on the fly.

Parameters & Return

$color: (string | list)

The name of a color in your palette, or a valid css color-value, with optional adjustments in the form of <origin-color> (<function>: <args...>).

$palette: $colors (Map)

A map containing the source color-palette for accessing previously-defined colors.

@return (Color)

A calculated css-ready color, based on your global color palette and adjustments.
