$text-ratios

scss

Description

A set of standard ratios for creating text rhythm and modular-scales.

Example

scss
// @preview ratios;
@include accoutrement-add('ratios', 'text-ratios', $text-ratios);

$spacing-sizes

  'double-gutter': 'gutter' ('linear': 2),
  'spacer': 'gutter' ('linear': 3),
  'double-spacer': 'spacer' ('linear': 2),
